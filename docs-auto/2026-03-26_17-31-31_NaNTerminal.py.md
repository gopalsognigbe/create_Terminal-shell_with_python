### Analyse du fichier `Terminal.py`

#### **Rôle du fichier**
Le fichier `Terminal.py` est un script Python qui simule un terminal interactif simple. Il permet à l'utilisateur d'exécuter des commandes système directement depuis l'interpréteur Python.

#### **Logique principale**
La logique principale est une boucle infinie (`while True`) qui :
1. Attend une commande de l'utilisateur via `input()`.
2. Gère spécifiquement la commande `cd` pour changer de répertoire.
3. Gère la commande `exit` pour quitter le terminal.
4. Exécute toutes les autres commandes via `subprocess.run()` et affiche les résultats.

#### **Fonctions importantes**
1. **`os.chdir()`** : Change le répertoire courant en fonction de l'argument fourni avec la commande `cd`.
2. **`subprocess.run()`** : Exécute une commande système et capture les sorties standard (`stdout`) et d'erreur (`stderr`).

#### **Exemple d'utilisation**
```
/home/user > ls
file1.txt  file2.txt  Documents
exit pour sortir

/home/user > cd Documents
/home/user/Documents > exit
```
Ce script permet d'interagir avec le système de fichiers et d'exécuter des commandes comme dans un terminal classique.
