### Analyse du fichier `Terminal.py`

#### Rôle du fichier  
Ce fichier implémente un terminal interactif simplifié. Il permet à l'utilisateur d'exécuter des commandes système directement depuis le script, tout en gérant les changements de répertoire et la sortie du terminal.

---

#### Logique principale  
Le fichier utilise une boucle infinie (`while True`) pour attendre les commandes de l'utilisateur. Il traite deux cas spécifiques :  
1. La commande `cd` pour changer de répertoire.  
2. La commande `exit` pour quitter le terminal.  
Pour toute autre commande, il utilise `subprocess.run` pour exécuter la commande système et afficher les résultats.

---

#### Fonctions importantes  
1. **`os.chdir()`** : Change le répertoire de travail actuel en fonction de l'argument passé.  
2. **`subprocess.run()`** : Exécute une commande système, capture sa sortie (stdout et stderr) et l'affiche.  
3. **`os.getcwd()`** : Récupère le répertoire de travail actuel pour l'afficher dans l'invite de commande.

---

#### Exemple d'utilisation  
1. Lancer le script :  
   ```bash  
   python Terminal.py  
   ```  
2. Utiliser des commandes :  
   ```bash  
   /home/user > ls  
   /home/user > cd Documents  
   /home/user/Documents > exit  
   ```  
   - `ls` liste les fichiers dans le répertoire actuel.  
   - `cd Documents` change le répertoire courant en "Documents".  
   - `exit` ferme le terminal.  

---

Ce fichier est utile pour créer un terminal minimaliste pour exécuter des commandes système dans un environnement Python.
