### Analyse du fichier `Terminal.py`

#### Rôle du fichier
Ce fichier implémente un terminal interactif simple qui permet à l'utilisateur d'exécuter des commandes système directement depuis un environnement Python. Il gère également des commandes spécifiques comme `cd` pour changer de répertoire et `exit` pour quitter le terminal.

#### Logique principale
Le script tourne en boucle infinie (`while True`), attendant que l'utilisateur entre une commande. Il traite la commande et exécute les actions appropriées :
1. **`cd`** : Change le répertoire courant en utilisant `os.chdir()`.
2. **`exit`** : Quitte la boucle et termine le programme.
3. **Autres commandes** : Exécute la commande via `subprocess.run()` et affiche la sortie standard (`stdout`) et les erreurs (`stderr`).

#### Fonctions importantes
1. **`os.chdir()`** : Change le répertoire courant.
2. **`subprocess.run()`** : Exécute une commande système et capture sa sortie et ses erreurs.
3. **`os.getcwd()`** : Récupère le répertoire courant pour l'afficher dans l'invite de commande.
4. **`input()`** : Lit l'entrée utilisateur.

#### Exemple d'utilisation
- Exécuter des commandes système comme `ls`, `dir`, `echo`, etc.
- Changer de répertoire avec `cd <dossier>`.
- Quitter le terminal avec `exit`.

Exemple d'interaction :
```
/home/user > ls
file1.txt file2.txt
exit pour sortir
/home/user > cd Documents
/home/user/Documents > 
```

Ce fichier est utile pour créer un environnement de ligne de commande interactif avec des fonctionnalités de base.
