### Analyse ultra-concise du fichier `Terminal.py` :

#### **Rôle**  
Terminal interactif basique pour naviguer/exécuter des commandes système.

#### **Cœur du système**  
1. Boucle infinie `cwd > [commande]`  
2. Deux types de commandes :  
   - Interne (`cd`) → gérée par Python  
   - Externe (`ls`, etc.) → passée à `subprocess.run(shell=True)` *(risque sécurité)*  

#### **Fonctions clés**  
- `change_directory()` : Gère la navigation  
- `execute_command()` : Lance les commandes système  

#### **Exemple**  
```bash
/home > cd /var/log  
/var/log > cat syslog
```

#### **Avertissement**  
Proof-of-concept éducatif uniquement. Jamais en production (shell=True dangereux).
