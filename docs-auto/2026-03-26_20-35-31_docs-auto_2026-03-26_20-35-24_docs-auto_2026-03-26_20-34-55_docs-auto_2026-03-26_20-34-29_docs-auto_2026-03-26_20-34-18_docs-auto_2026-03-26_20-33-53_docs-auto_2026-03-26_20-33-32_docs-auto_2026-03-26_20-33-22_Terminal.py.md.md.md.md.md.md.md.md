### Analyse ultra-concise du fichier `Terminal.py` :

**Rôle** : Shell minimaliste avec gestion de répertoire (REPL interactif).

**Logique** :  
1. Boucle infinie (`exit` pour quitter)  
2. Détection spéciale de `cd`, autres commandes via `subprocess.run` (attention: `shell=True` non sécurisé)

**Fonctions clés** :  
- `change_directory()` : navigation sécurisée  
- `execute_command()` : exécution système + gestion erreurs  

**Exemple** :  
```bash
~/projet $ ls
~/projet $ cd src
~/projet/src $ exit
```

**À retenir** :  
- Extensible pour commandes custom  
- Risque sécurité à corriger (`shell=False` + liste d'arguments)
